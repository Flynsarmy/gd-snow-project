# Godot 4 Snow Displacement

Raises snow up based on terrain vertex colours, then stomps it back down under a character's feet as they move around.

Taken mostly from [this video](https://www.youtube.com/watch?v=BXo97H55EhA), adapted for Godot 4.7 making use of its new [DrawableTextures](https://docs.godotengine.org/en/stable/tutorials/rendering/drawable_textures.html)
